Level 2 BTEC in Creative Technology and IT



The BTEC ICT unit 2 Technology Systems course covers all topics from the specification.

These topics are:

Learning Aim A

- Understand how the components of technology systems work together.
- Computers and Tech Systems
- Applications and Issues of Tech Systems
- Computer Hardware Devices
- Computer Networking
- Data Transfer

Learning aim B:

- Understand how data flows between internal components of a computer and is processed to provide information.
- Internal Components of a Computer
- Analogue and Digital Data

Learning aim C:

- Understand different types of software.
- Software
- Introduction to Computer Programming Concepts
- Operating Systems and Applications
